# Hash-Table-OpenMP

### Краткое описание
Результат вычисления hash-суммы по данным однозначно указывает на адрес, по которому они хранятся. Позволяет извлекать данные очень быстро, но неэффективно использует свободное пространство
### Компоненты
Алгоритм построения hash-таблицы<br>
Алгоритм поиска данных в hash-таблице
